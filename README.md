# Guia do Aprendiz - Lojas Becker

## Sobre o Projeto

Web app educativo desenvolvido como protótipo de sistema de integração para o Programa Jovem Aprendiz.

Este projeto simula um portal corporativo onde novos aprendizes podem encontrar todas as informações necessárias para sua adaptação e crescimento dentro da empresa. A plataforma centraliza orientações sobre rotinas, ferramentas essenciais e diretrizes organizacionais em uma interface intuitiva e de fácil acesso.

## Objetivo

Criar uma referência de como a tecnologia web pode otimizar processos de onboarding e treinamento, oferecendo aos jovens aprendizes um guia completo e interativo para seu desenvolvimento profissional.

## Funcionalidades

- Sistema de autenticação para conteúdo restrito
- Guia de tarefas diárias com cards expansíveis
- Tutoriais de ferramentas (Word, Excel, sistemas internos)
- FAQ interativo com dúvidas frequentes
- Informações institucionais sobre a empresa
